# Outdoor Afro Dynamic Fund Raising Thermometer

A dynamic exhibit that allows Outdoor Afro (https://outdoorafro.org/) to update fund raising progress against annual goal for review with Board of Directors

This version utlilizes a simple server that makes the API call to TMDB as a way of masking token on the client-side.

## Live Demo
[Play SnoMan](https://trinidads-portfolio.com/SnoMan/index.html)

## Features
- Dynamic exhibit as text entries are updated
- Dynamic text formatting based on raised thresholds
- Ability to Copy exhibit for direct insertion into PowerPoint or Download a version
- Local storage to maintain users last entries

## Setup Instructions

1. Clone this repository
2. Open `index.html` in your browser

## Technologies Used
- HTML5
- Vanilla JavaScript (ES6 modules)
- CSS3 with responsive design
- Local Storage API
- SVG for thermometer
